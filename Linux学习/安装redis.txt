1、下载redis安装包
下载链接  http://redis.io/download
$ wget http://download.redis.io/releases/redis-2.8.17.tar.gz

2、解压压缩包
$ tar xzf redis-2.8.17.tar.gz
$ cd redis-2.8.17

3、安装redis
$ make

4、启动redis
$ cd src
$ ./redis-server

默认配置启动redis
$ cd src
$ ./redis-server ../redis.conf


5、操作 redis

$ ./redis-cli
redis> set foo bar
OK
redis> get foo
"bar"
